## Core Features:

- Dumpster Registration: Enable the user to register their dumpsters with unique identifiers and set their initial status (Available, Rented, In Maintenance).
- Client Management: Maintain a client directory with essential contact information (Name, Phone, Address).
- Rental Logging: Log new rentals by selecting a dumpster, assigning it to a client, specifying the delivery address, and setting rental dates. Automatically update the dumpster status to 'Rented'.
- Dashboard Overview: Provide a dashboard to view all currently rented dumpsters, including details such as dumpster ID, client name, location, and expected return date. Offer a quick action to mark a rental as complete.

## Style Guidelines:

- Primary color: A muted teal (#70A4A4), evoking cleanliness and reliability.
- Background color: Very light gray (#F0F0F0), offering a clean and unobtrusive backdrop.
- Accent color: Pale gold (#B4A470) to provide a sense of simple quality, utility, and durability.
- Font pairing: 'Inter' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, providing a balance of modern clarity and readability.
- Use clear and simple icons to represent different dumpster statuses and actions (e.g., a truck for delivery, a calendar for due date). Line icons would complement the overall minimalistic design.
- A clean, card-based layout for displaying dumpster information on the dashboard, ensuring information is easily digestible. Prioritize key details such as client, location, and due date.
- Subtle transitions when updating dumpster statuses or navigating between sections to provide smooth user experience without being distracting.
